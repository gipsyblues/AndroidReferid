########################################################################
# MIFARE CLASSIC TOOL DUMP TO RAW (HEX) DUMP - Dump Converter          #
########################################################################

The script "mct-dump2raw-dump.sh" converts a dump file generated by MCT
into a raw hex dump.

Dependencies:
- perl
- sed ;)

########################################################################

Usage: ./mct-dump2raw-dump.sh <dumpFile>

dump-file:
  The dump file is a file that was generated by the MCT-App.
  This is done by reading a full tag and then saving whole data.
  It should look like the "example-dump-file.txt"
  (see "example-files" directory).
  You can find the files in "MifareClassicTool/dump-files/" of your
  external storage (SD-Card).

Usage Examples:
  ./mct-dump2raw-dump.sh example-dump-file.txt > hexdump.raw
