########################################################################
# RAW (HEX) DUMP TO MIFARE CLASSIC TOOL DUMP - Dump Converter          #
########################################################################

The script "raw-dump2mct-dump.sh" converts a raw (hex) dump file into
a MCT dump file.

Dependencies:
- xxd

########################################################################

Usage: ./raw-dump2mct-dump.sh <dumpFile>

dump-file:
  A raw dump file (e.g. generated by libnfc).
  It should look like the "example-dump-file.raw"
  (see "example-files" directory).

Usage Examples:
  ./raw-dump2mct-dump.sh example-dump-file.raw > mct-dump
